# SQL-Project-for-sales-revenue-
 sales &amp; revenue analytics query
 Project Overview
This project demonstrates SQL skills using a dummy sales dataset.
It includes customer, order, order_items, and product tables.

 Database Schema
- customers
- orders
- order_items
- products

 SQL Analysis Performed
1. Top 10 customers by revenue
2. Revenue by product category
3. Monthly sales trend

Tools Used
- SQL (MySQL / PostgreSQL / SQL Server)
- GitHub

 How to Run
1. Run `schema/create_tables.sql`
2. Run `data/insert_dummy_data.sql`
3. Execute queries inside `queries/`

 Key Learnings
- JOINs
- GROUP BY & Aggregations
- Date functions
- Business analysis using SQL
